# Domain Transformer (Nano)

GPT-2 small from scratch in PyTorch, trained on domain-specific corpus. Not a fork — written independently to prove understanding of every layer.

## Stack
PyTorch, tiktoken, WandB, mixed precision training

## Dataset
Insurance claims descriptions + policy documents

## Metrics Target
- Val perplexity < 30
- Throughput > 50K tokens/sec (T4)
